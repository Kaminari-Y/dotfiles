# Homebrew
## Install xcode
* Install xcode from Appstore
* Install Command Line Tools
```
xcode-select --install
```

## Install Homebrew
Use the command below
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew -v
```
